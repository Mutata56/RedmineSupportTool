# <название> Support Pro

Расширение для Chrome, которое объединяет работу с заявками Redmine, мониторинг почты и чатов в одном месте. Написано для ежедневного использования командой техподдержки.

> **Почему плейсхолдеры?** Расширение используется в продакшне. Название, ссылки на Redmine, названия проектов и другие внутренние данные заменены на заглушки (`<название>`, `<ссылка>`, `<project_id>` и т.д.), чтобы не раскрывать информацию о компании и инфраструктуре.

---

## Что умеет

### Канбан-дашборд

Доска заявок Redmine с тремя колонками: **Новая**, **В работе**, **Ожидание**. Показывает общее количество, задачи с высоким приоритетом и на удержании. Есть сортировка, фильтры по времени и автообновление. Новые заявки сопровождаются звуковым сигналом.

<img width="2328" height="1419" alt="image" src="https://github.com/user-attachments/assets/8418a981-2a09-4a89-bfdc-2c58ba850648" />


### Мониторинг Gmail

Отслеживает непрочитанные письма по настраиваемому запросу (HealthScript, Notifier). Каждое письмо можно превратить в заявку Redmine одной кнопкой. Фильтр «тихих пунктов» - письма с определёнными номерами расхождений не триггерят сигнал.

### Мониторинг Яндекс Почты

Плавающая кнопка «Создать заявку» над письмами с расхождениями WMS. Автоматически подбирает проект Redmine по отправителю или имени сервера.

### Мониторинг Яндекс Мессенджера

Следит за непрочитанными сообщениями, обновляет счётчик на иконке расширения.

### Мониторинг корпоративного мессенджера

Отслеживает канал на наличие ключевых слов: ЧС, ЧП, критичность, массовые инциденты, обращения к команде. При срабатывании - метит пост и подаёт сигнал.

### Звуковой будильник

Повторяющийся звук при новой заявке или критическом сообщении. Останавливается кнопкой «Стоп» или подтверждением.

### Счётчик на иконке

Агрегирует события из всех источников: Redmine, Gmail, Мессенджер, <Плейсхолдер>. Красный - есть срочные задачи.

### SQL-библиотека

Хранилище SQL-скриптов с добавлением, поиском, фильтрацией по клиенту, импортом/экспортом в JSON и подсветкой синтаксиса.

### Контекстное меню на заявках

Правый клик на странице заявки:

- **Взять в работу** - статус + назначение на себя
- **Добавить в смену** - с проверками (заполнение форм причины обращения и трудозатрат)
- **Отформатировать информацию о заявке для быстрой отправки консультантам** - ссылка + описание в буфер
- **Подтянуть скрипт HealthScript** - SQL из wiki Redmine
- **Быстрые ссылки** - Графана, LOCK-запросы, статистика

### Поиск похожих заявок

Автоматический поиск похожих задач по ключевым словам из темы/описания и кодам ошибок. Результаты с фрагментами решений. Фильтр по подпроектам клиента.

### LogViewer

Просмотр лог-файлов (.log4j, .log, .xml) в браузере. Таблица с фильтрами по уровню, потоку, пользователю, времени и тексту. Закладки, группировка по транзакциям, визуальная шкала времени.

## Установка

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку с расширением

---

## Требования

- Google Chrome (Manifest V3)
- Доступ к `<ссылка>` (Redmine)
- Google OAuth токен для Gmail API
